The resources for CSS in W3Schools are short for D3.
CSS element Selector
(https://www.w3schools.com/cssref/sel_element.asp)
You can select and style all <p> elements.

CSS element,element Selector
(https://www.w3schools.com/cssref/sel_element_comma.asp)
You can select and style all <h2> and all <p> elemetns by seperating them with a comma in a stylesheet.

CSS element element Selector
(https://www.w3schools.com/cssref/sel_element_element.asp)
"Select and style every <p> element that is inside <div> elements" by typing "div p {" without the double quotations.

CSS .class Selector
(https://www.w3schools.com/cssref/sel_class.asp)
Ex.
.class {

}
Select and style all elements with class labeled "intro."

Ex. 1
p.hometown {
  background-color: yellow;
}
Style all <p> elements with "hometown" class.

Ex. 2
<p class="center big">This paragraph refers to two classes.</p>
This <p> element will be styled to class "center" and "big."
